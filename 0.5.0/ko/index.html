FuriosaAI NPU 및 Software 0.5.0 문서

이 문서는 FuriosaAI NPU 소개와 NPU 활용에 필요한 Software 설치 및 사용 방법을 제공한다.

참고

FuriosaAI의 소프트웨어는 커널 드라이버, 펌웨어 및 런타임 및 C SDK, Python SDK 및 명령행 도구를 포함한다. FuriosaAI의 소프트웨어는 평가 프로그램(Early Access Program) 등록 후 최종 사용자 라이센스 동의(End User License Agreement)에 따라 배포되며, 현재는 contact@furiosa.ai 로 문의하여 평가 프로그램을 시작할 수 있다.
