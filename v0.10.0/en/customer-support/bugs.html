Bug ReportÔÉÅ

If you encounter an unresolvable issue, you can file a bug report at FuriosaAI customer service center. The following information should be included in a bug report.

  1. How to reproduce the bug

  2. Log or screenshot of the bug

  3. SDK version information

  4. Compilation log, if model compilation failed

By default, when an error happens furiosa-sdk outputs the following message. If you see the following message, file a report to the Bug Report section of FuriosaAI customer service center with:

  1. The information given below the Information Dump, and

  2. The compilation log file (this would be /home/furiosa/.local/state/furiosa/logs/compile-20211121223028-l5w4g6.log in the following example) outputted in the message.
